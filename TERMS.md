## Privacy

Key54 runs 100% locally, collects no data, and has no analytics or
telemetry. The only network request it makes is a version check against
GitHub Releases to offer updates. See the full
[Privacy Policy](PRIVACY.md).

## Updates

Key54 checks GitHub for new releases and can update itself (via
Homebrew, if installed that way, or by pointing you at the latest
release). You can simply not install updates if you prefer.
