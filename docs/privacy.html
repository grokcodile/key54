Privacy Policy

Effective July 2, 2026

The short version: Key54 collects no data. There are no accounts, no analytics, and no tracking — in the app or on this site. Everything you configure stays on your Mac.

The app

  • Key54 has no account system and no telemetry. It never collects, stores, or transmits personal information.
  • Your settings (the app you bind, hold duration, animation style) are stored locally on your Mac and never leave it.
  • The Accessibility permission macOS asks for is used for one thing: observing the right Command (⌘) key so the trigger works. It is listen-only — Key54 does not record, log, or transmit any keystrokes.
  • The app's only network activity is checking GitHub's public API for a newer release, and downloading one if you choose to update. These are ordinary web requests with no personal identifiers added; GitHub's standard server logging applies.

This website

  • key54.app is a static site with no cookies, no analytics, and no trackers.
  • It is hosted on GitHub Pages, which — like any web host — may keep standard server logs (such as IP addresses) per GitHub's privacy statement.

Third-party services

If you choose to use them, these services have their own privacy policies:

  • GitHub — hosts the code, releases, and this site, and serves the update check.
  • Homebrew — if you install or update via brew.
  • Ko-fi — if you leave a tip. Key54 never sees any payment details.

Changes

If this policy ever changes, the update will be posted on this page with a new effective date. Given what Key54 is — a tiny, open-source utility — don't expect much to change.
